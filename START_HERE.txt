START HERE

1. Install Node.js LTS.
2. Open this folder in VS Code.
3. Run: npm install
4. Copy apps/api/.env.example to apps/api/.env
5. Put your OpenAI API key inside apps/api/.env
6. Run: npm run dev
7. Open http://localhost:5173
8. Paste your repo folder path into the app.
9. Ask: Analyze this repo, find errors, fix them, and run tests.

Important: direct folder editing must run locally. Cloud mode needs GitHub or ZIP upload.
